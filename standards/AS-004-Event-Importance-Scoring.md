\# 1. Purpose



This Standard defines how Atlas measures event importance.



Importance is determined by impact,



not media attention.



\---



\# 2. Scoring Dimensions



Each event is evaluated in five dimensions.



\---



\## Market Impact



Does the event affect the market?



Score:



0–5



\---



\## Company Impact



How many companies are affected?



Score:



0–5



\---



\## Industry Impact



How many industries are affected?



Score:



0–5



\---



\## Time Horizon



How long will the impact last?



Score:



0–5



\---



\## Historical Significance



Will this event still matter in five years?



Score:



0–5

\# 5. Atlas Principle



Media attention



≠



Knowledge importance.



Atlas records importance,



not popularity.



\# AI Enforcement



Before inserting any event into Atlas, the AI must verify:



□ Event has objectively occurred



□ Source is verified



□ Event meets quality standard



□ Importance score calculated



□ Related entities identified



□ Event ID assigned



Only after all checks pass may the event enter the Atlas Knowledge Graph.
