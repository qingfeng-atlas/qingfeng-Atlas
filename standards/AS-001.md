# AS-001

# 资本市场事件生命周期标准

Version: 0.1

Date: 2026-07-09


## 1. Purpose

建立统一的资本市场事件记录标准。

Atlas记录的不是新闻，而是事件。


## 2. Core Principle

每一个事件都需要：

- 来源
- 时间
- 关联对象
- 影响
- 后续结果


## 3. Event Lifecycle

发现

↓

确认

↓

传播

↓

兑现

↓

历史沉淀


## 4. Evidence Rule

所有Atlas结论必须有证据支持。

不记录猜测。

不制造观点。


## 5. Philosophy

每一个事件都值得记录。

每一个结果都值得验证。
