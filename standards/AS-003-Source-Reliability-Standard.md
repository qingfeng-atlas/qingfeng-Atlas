\# AS-003 Source Reliability Standard



Version: 1.0



Status: Active



Owner: Project Atlas



Created: 2026-07-11



\---



\# 1. Purpose



This Standard defines how Atlas evaluates information sources.



Atlas trusts evidence,



not popularity.



\---



\# 2. Reliability Levels



\## Level A — Primary Sources



Highest credibility.



Examples:



• State Council



• CSRC



• National Bureau of Statistics



• Shanghai Stock Exchange



• Shenzhen Stock Exchange



• Listed Company Announcements



Trust Score:



★★★★★



\---



\## Level B — Official Institutions



Examples:



• Government Departments



• Industry Associations



• Central Bank



• Customs



Trust Score:



★★★★☆



\---



\## Level C — Professional Media



Examples:



• Xinhua



• Securities Times



• China Securities Journal



• Caixin



Trust Score:



★★★☆☆



\---



\## Level D — Secondary Sources



Examples:



• Financial Blogs



• Self-media



• Market Commentary



Trust Score:



★★☆☆☆



\---



\## Level E — Unverified Information



Examples:



• Rumors



• Chat Groups



• Anonymous Posts



Trust Score:



★☆☆☆☆



Default Action:



Reject



\---



\# 3. Atlas Rule



If multiple sources conflict,



Atlas always prefers



the highest reliability source.



\---



\# 4. AI Requirement



Every AI-generated explanation



must display



Source Level.



\# AI Enforcement



Before inserting any event into Atlas, the AI must verify:



□ Event has objectively occurred



□ Source is verified



□ Event meets quality standard



□ Importance score calculated



□ Related entities identified



□ Event ID assigned



Only after all checks pass may the event enter the Atlas Knowledge Graph.
